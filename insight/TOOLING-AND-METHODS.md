Supporting tools are considered XP enablers rather than XP practices.

## Development Methods

### Extreme Programming (XP)

Primary development methodology.

### Trunk-Based Development

Primary branching strategy.

Characteristics:

- main as integration branch
- short-lived branches
- rapid integration
- simple history

### No Estimates

Planning is based on:

- small increments
- frequent feedback
- continuous prioritization
- delivered value

instead of effort estimation.
